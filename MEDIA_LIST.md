## RECOMMENDED IMAGE SPECIFICATIONS

### Static Images (WebP)
- **Quality:** High quality, 80-90% compression
- **Resolution:** 2x for retina displays
  - Feature cards: 640x853px (1280x1706 @2x)
  - Blog cards: 640x853px (1280x1706 @2x)
  - Certification previews: 1280x720px (2560x1440 @2x)
  - Testimonial posters: 640x1138px (1280x2276 @2x)
  - Hero images: 1920x1080px (3840x2160 @2x)
- **Color profile:** sRGB
- **Format:** WebP for all images (better compression than JPG)

### Videos (WebM)
- **Resolution:** 1920x1080 or 1080x1920 (portrait)
- **Codec:** VP9 (WebM)
- **Bitrate:** 5-8 Mbps for quality, consider compressed versions for mobile
- **Audio:** Opus 128kbps (for testimonials), muted for hero
- **Frame rate:** 24fps or 30fps
- **Duration:** Hero 10-15s loop, Testimonials 30-45s each
